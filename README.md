# Todolist
->>create the project :
python-admin startproject Projectname
->>then create app:
py manage.py startapp appname

->>create the environment:
py -m venv envname
then ,
cd envname/Scripts
activate

->> make the change in models.py 
Then run,
py manage.py makemigrations
py manage.py migrate


->> run the server
py manage.py runserver
